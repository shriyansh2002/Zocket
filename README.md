# AI Task Manager
A real-time task management system built in 4 hours.

## Features
- JWT Auth
- Task CRUD
- Real-time Updates (WebSockets)
- AI Suggestions (Gemini API)

## Setup
1. Backend:
   - cd backend
   - go mod init ai-task-manager
   - go get dependencies listed in main.go
   - Set up PostgreSQL: CREATE DATABASE task_manager; and run schema from docs
   - go run main.go
2. Frontend:
   - cd frontend
   - 
pm install
   - 
pm run dev
